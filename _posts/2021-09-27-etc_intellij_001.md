- 테스트 코드로 이동, 테스트 코드 생성
    - `Ctrl` + `Shift` + `T`
- Optimize Imports
    - `Ctrl` + `Alt` + `O`
- 변수를 이용한 리펙토링
    - `Ctrl`+ `Alt` + `V` 

- 방금 실행한 코드 다시 실행
    - `Shift` + `F10`
- 영역에 따라 Test 코드 실행
    - `Ctrl` + `Shift` + `F10`
