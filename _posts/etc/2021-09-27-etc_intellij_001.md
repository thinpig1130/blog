- 테스트 코드로 이동, 테스트 코드 생성
    - `Ctrl` + `Shift` + `T`
- Optimize Imports
    - `Ctrl` + `Alt` + `O`
- 변수를 이용한 리펙토링
    - `Ctrl`+ `Alt` + `V` 

- 방금 실행한 코드 다시 실행
    - `Shift` + `F10`
- 영역에 따라 Test 코드 실행
    - `Ctrl` + `Shift` + `F10` 


- 어떤 파라미터를 입력해야 하는 차례인지... 팝업을 통해 알려줌.
    - `Ctrl` + `P`

- 프로젝트 목록 으로 pocus 이동
    - `Alt`+ `1`

- 줄바꿈
    - `Ctrl` + `Shift` + ` ↑ ` OR ` ↓ `


- 변수명 또는 Class명 변경 (사용된 것까지 모두 변경)
    - `Shift` + `F6`
